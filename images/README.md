# FreshPress Hero Image

This directory contains images for the FreshPress Dry Cleaners website.

## Required Images:

1. **hero.jpg** (1920x1080px recommended)
   - Professional dry cleaning shop interior or clean garments
   - Should convey quality and cleanliness
   - Used as background for the hero section

## Suggested Image Sources:
- Unsplash.com (search: "dry cleaning", "laundry", "professional cleaning")
- Pexels.com (search: "clean clothes", "garment care")
- Stock photo services

## Image Guidelines:
- High quality, professional appearance
- Good contrast for text overlay
- Conveys trust and cleanliness
- Minimum resolution: 1200x800px
- File format: JPG or PNG
- File size: Under 500KB for web optimization

## Note:
The website will work without images, but adding professional photos will greatly enhance the visual appeal.
